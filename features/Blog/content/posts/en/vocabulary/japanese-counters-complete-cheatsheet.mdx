In English, we count "one apple, two apples, three apples." Simple. In Japanese, you count "apple one-THING, apple two-THING, apple three-THING"—and the "THING" changes depending on what you're counting.

Welcome to Japanese counters (助数詞 - josūshi), where flat things, long things, small animals, and machinery all get their own special counting words.

<InfoBox type='note'>
  There are over 500 counters in Japanese, but don't panic! About 20 cover 95%
  of daily situations. Master these, and you'll handle almost anything.
</InfoBox>

## The Universal Escape Counter: つ

When you don't know the right counter, use the **native Japanese counting system** with つ:

| Number | Counter  | Usage           |
| ------ | -------- | --------------- |
| 1      | ひとつ   | General objects |
| 2      | ふたつ   | General objects |
| 3      | みっつ   | General objects |
| 4      | よっつ   | General objects |
| 5      | いつつ   | General objects |
| 6      | むっつ   | General objects |
| 7      | ななつ   | General objects |
| 8      | やっつ   | General objects |
| 9      | ここのつ | General objects |
| 10     | とお     | General objects |

**After 10:** Switch to the Chinese-derived system with specific counters or just say the number.

<InfoBox type='tip'>
  The つ counter only works up to 10! For 11+, you need proper counters or use
  個 (ko) as a general fallback.
</InfoBox>

## Essential Counters by Category

### 人 (にん/り) — People

The most important counter! Note the irregular readings:

| Number   | Reading                     | Example         |
| -------- | --------------------------- | --------------- |
| 1 person | ひと**り**                  | 一人            |
| 2 people | ふた**り**                  | 二人            |
| 3 people | さん**にん**                | 三人            |
| 4 people | よ**にん**                  | 四人            |
| 5+       | ご**にん**, ろく**にん**... | Regular pattern |

### 個 (こ) — Small Objects (General)

Use for small, roundish, or compact items when you don't know the specific counter:

| Number | Reading      |
| ------ | ------------ |
| 1      | いっ**こ**   |
| 2      | に**こ**     |
| 3      | さん**こ**   |
| 6      | ろっ**こ**   |
| 10     | じゅっ**こ** |

**Use for:** Apples, eggs, balls, erasers, stones, boxes, gifts

### 本 (ほん) — Long, Thin Objects

For anything long and cylindrical:

| Number | Reading      |
| ------ | ------------ |
| 1      | いっ**ぽん** |
| 2      | に**ほん**   |
| 3      | さん**ぼん** |
| 4      | よん**ほん** |
| 6      | ろっ**ぽん** |

**Use for:** Pens, pencils, bottles, trees, umbrellas, bananas, roads, rivers, fingers

<InfoBox type='warning'>
  Notice the sound changes! 本 becomes ぽん, ぼん, or ほん depending on the
  number. This is called 音便 (onbin).
</InfoBox>

### 枚 (まい) — Flat Objects

For thin, flat things:

| Number | Reading      |
| ------ | ------------ |
| 1      | いち**まい** |
| 2      | に**まい**   |
| 3      | さん**まい** |

**Use for:** Paper, tickets, photos, stamps, plates, shirts, blankets, CDs, pizza (!)

### 匹 (ひき) — Small Animals

For small to medium animals (not birds, not humans):

| Number | Reading        |
| ------ | -------------- |
| 1      | いっ**ぴき**   |
| 2      | に**ひき**     |
| 3      | さん**びき**   |
| 6      | ろっ**ぴき**   |
| 10     | じゅっ**ぴき** |

**Use for:** Dogs, cats, fish, insects, mice, rabbits

### 頭 (とう) — Large Animals

For large animals:

**Use for:** Cows, horses, elephants, whales, lions

> 動物園に象が**三頭**います。
> (The zoo has three elephants.)

### 羽 (わ) — Birds and Rabbits

Yes, rabbits use the bird counter! Historical quirk.

| Number | Reading                 |
| ------ | ----------------------- |
| 1      | いち**わ**              |
| 3      | さん**ば** / さん**わ** |

**Use for:** Birds, chickens, rabbits

### 台 (だい) — Machines and Vehicles

For mechanical things:

**Use for:** Cars, computers, TVs, refrigerators, bicycles, beds

> 車を**二台**持っています。
> (I have two cars.)

### 杯 (はい) — Cups and Glasses

For drinks in containers:

| Number | Reading      |
| ------ | ------------ |
| 1      | いっ**ぱい** |
| 3      | さん**ばい** |

**Use for:** Cups of coffee, glasses of water, bowls of rice/noodles

### 冊 (さつ) — Bound Volumes

For books, magazines, notebooks:

| Number | Reading      |
| ------ | ------------ |
| 1      | いっ**さつ** |
| 8      | はっ**さつ** |

### 階 (かい) — Floors/Stories

For building floors:

| Number | Reading                     |
| ------ | --------------------------- |
| 1F     | いっ**かい**                |
| 3F     | さん**がい**                |
| 8F     | はっ**かい** / はち**かい** |

> エレベーターで**五階**に行きます。
> (I'll go to the 5th floor by elevator.)

## Time Counters

### 時 (じ) — O'clock

| Time | Reading    |
| ---- | ---------- |
| 1:00 | いち**じ** |
| 4:00 | よ**じ**   |
| 7:00 | しち**じ** |
| 9:00 | く**じ**   |

## Sound Change Patterns (音便)

Many counters change sounds. Here's the pattern:

| Number      | Changes to...               |
| ----------- | --------------------------- |
| 1, 6, 8, 10 | Often double consonant (っ) |
| 3           | Often voiced (b, g, d)      |
| What        | なん + counter              |

**Example with 本:**

- 1本 = いっ**ぽ**ん (h → p)
- 3本 = さん**ぼ**ん (h → b)
- 6本 = ろっ**ぽ**ん (h → p)

## Common Mistakes

### Mistake 1: Using つ for People

❌ 三つの学生 (sounds like counting objects)
✅ 三人の学生

### Mistake 2: Wrong Animal Counter

❌ 犬が三頭います (頭 is for large animals)
✅ 犬が三匹います

### Mistake 3: Forgetting Sound Changes

❌ 一ほん, 三ほん
✅ 一**ぽ**ん, 三**ぼ**ん
